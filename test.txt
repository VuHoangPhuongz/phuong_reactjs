Học Git cơ bản
Github

I. 3 bước để đưa source code lên git remote
1. Xác định các file hoặc folder cần đẩy lên git
git add [path]
path: 1 đường dẫn, đến tên file hoặc tên folder

git add .
Trong project có tất cả bao nhiêu file bị thay đổi (thêm mới, sửa, xoá) và add vào.

2. Mô tả cho mọi người biết mình vừa thay đổi cái gì
git commit -m

3. Đẩy code lên server
git push
